## Link in Bio

This is my personal link-in-bio webpage. It includes my profile image and links to my social and professional accounts. I built it using basic HTML and CSS.

## How to View the Project

1. Download this project to your computer.
2. Open the project folder.
3. Double-click the `index.html` file to open it in your web browser.
